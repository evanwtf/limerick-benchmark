# Task: Limerick Generator

Create a Python web application that generates and displays random limericks in the browser.

## Requirements

- Use **Flask** for the web framework
- Use **uv** for package management (`uv init`, `uv add`, `uv run`)
- The app must listen on **port 8181**
- The page must display a randomly generated limerick
- The page must automatically show a **new limerick every 5 seconds** without requiring the user to refresh
- The limericks should be original — generated by the app, not loaded from a static list (though a static list as a fallback is fine)

## Limerick format

A limerick is a 5-line poem with the AABBA rhyme scheme:
- Lines 1, 2, 5: longer, rhyme together (A)
- Lines 3, 4: shorter, rhyme together (B)

Example:
```
There once was a man from Nantucket
Who kept all his cash in a bucket.
    But his daughter, named Nan,
    Ran away with a man
And as for the bucket, Nantucket.
```

## Deliverable

A working application that can be started with `uv run` from the project directory. The exact entry point is up to you (`app.py`, `main.py`, etc.) but document it clearly.

## Constraints

- No external APIs — limericks must be generated locally
- Must work on macOS (Apple Silicon)
- Python 3.11+

## Hints

- `uv init .` initializes in the current directory
- `uv add flask` adds Flask as a dependency
- You can use JavaScript `setInterval` or a `<meta http-equiv="refresh">` tag for auto-refresh
- Test that the server actually starts and responds before declaring done
